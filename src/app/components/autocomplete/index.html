The autocomplete component allows users to search and select items.

Specifies the available items to select. Specifies a function to apply when text changes to dynamically manage the data source.
Acceptable value: (searchText: string, data: any[]) => Array or Promise Array. Specifies an array of functions that are called against each search result. Return true or false to display the result in the dropdown.
Acceptable value: (searchText: string, item: any) => boolean. The minimum number of characters the user must enter before a search is performed. Indicates an array of object properties to search. Indicates the property of the object to use as the display value in the input field after it is selected from the dropdown. The maximum number of results shown in the drop down. If no limit is specified, all matching results will be displayed. Specifies a template to be used for each search result. The template has access to the item variable references the search result's object value. Fires when a selection is made and provides an argument of type SkyAutocompleteSelectionChange (see below). References the selected search result object.
