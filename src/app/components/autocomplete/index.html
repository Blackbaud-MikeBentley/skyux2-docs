The autocomplete component allows users to search and select items.

Specifies the available items to select. Specifies a function to apply when text changes to dynamically manage the data source Acceptable value: (searchText: string, data: any[]) => Array or Promise Array. Specifies an array of functions that are called against each search result. Return true or false to display the result in the dropdown. Acceptable value: (searchText: string, item: any) => boolean. The minimum number of characters the user must enter before a search is performed. Indicates an array of object properties to search. Indicates the property of the object to use as a descriptor when the item is selected. The maximum number of results shown in the drop down. If no limit is specified, all matching results will be displayed. Specifies a template for search results. This can be assigned as a reference to the template attribute, or it can be assigned as a child of the template element inside the sky-lookup component. The template has access to the item variable which contains the object to represent. Fires when a selection is made and provides an argument of type SkyAutocompleteSelectionChange (see below).
